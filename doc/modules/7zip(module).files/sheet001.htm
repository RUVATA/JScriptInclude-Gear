Explication: Этот Метод  устанавливает режим отображения вывода архиватора 7za.exe в консоль
jlib7zaArchivator.setDisplayMode(mode) {Method}
Arguments:
{String} mode * Выбранный режим отображения.           RUS
"visible" * Вывод отображается. Окно консоли с системными параметрамиокана окна. RUS
"hidden" * Вывод не отображается.             RUS
"minimazed" * Вывод отображается. Окно консоли в свернутом окне.     RUS
"maximazed" * Вывод отображается. Окно консоли в максимально развернутом окне. RUS
Default:
{String} "visible"
Внимание!!! Значение по умолчанию, для режима отображения вывода архиватора в консоль  RUS
определяется в конструкторе класса, и от вызова этого метода не зависит.
  CONTENT ↑
Explication: Этот Метод  включает файл или каталог в список  для предстоящей архивации.
jlib7zaArchivator.toList(targetFile) {Method}
Arguments:
{String} targetFile * Абсолютный или относительный путь к целевому файлу или каталогу. RUS
Внимание!!! Предстоящее добавление каталогов в архив будет происходить по правилам  RUS
предписанным архиватором 7zip, т.е. каталоги пути к которым указанны без
завершающющего символа разделения каталогов "\\"( backslash) - будут
включены в архив целиком, в противном случае в архив бкдет включено 
лишь содержимое этого каталога.          
Return:
{Number} 0 * Непредвиденная ошибка.             RUS
{Number} -1 * Оперция произвежена успешно.               RUS

Explication: Этот Метод запускает процесс архивации.          
jlib7zaArchivator.Archivate(TargetFile, ArchiveName) {Method}
Arguments:
{String} TargetFile * Абсолютный или относительный путь к целевому файлу или каталогу. RUS
Внимание!!! Предстоящее добавление каталогов в архив будет происходить по правилам  RUS
предписанным архиватором 7zip, т.е. каталоги пути к которым указанны без
завершающющего символа разделения каталогов "\\"( backslash) - будут
включены в архив целиком, в противном случае в архив бкдет включено 
лишь содержимое этого каталога.          
{String} ArchiveName * Имя будующего архива, с возможностью использования макро-вставок RUS
Доступные макро-вставки:             RUS
"YYYY"  - преобразуется в текущий год в соответвующем формате.  
"YY"  - преобразуется в текущий год в соответвующем формате.  
"MM"  - преобразуется в текущий месяц в соответвующем формате.  
"DD"  - преобразуется в текущий день месяца в соответвующем формате.
Внимание!!! Подмена макровставок производится при точном совпадении с шаблоном, RUS
все встречающиеся совпадения будут подменены.

JScriptInclude(sureFSO) Модуль - Надстройка над ActiveXObject('Scripting.FileSystemObject')
Предназначен для поддержки режима игнорирования и логирования исключений при работе с его базовыми методами, отображения более детальной информации о возникшем исключении.

ВНИМАНИЕ:
Данный объект синтаксически копирует методы и свойства обычного 
экземляра ActiveXObject('Scripting.FileSystemObject'), но на данный момент
поддерживаются лишь методы: FolderExist, FileExist, CopyFile, CopyFolder,
CreateFolder, DeleteFile, DeleteFolder.
Модуль не окончен, скорее всего будет переписываьть с использованием

//  [ENG]

FSO.setOnErrorsMode("ignore");          
                   
FSO.DeleteFile(newArch + "[ENG]this is done to raise an exception /" + 
"[RUS]Это сделано специально чтобы вызвать исключение");    
FSO.DeleteFile(newArch);

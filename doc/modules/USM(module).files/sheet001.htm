Explication: Этот метод проверяет строку на предмет наличия в ней макровставок, и в случае
обнаружения произволдит соответсвующую замену.        
USM.MacrosBase.MacrosParser(MacroString) {Method}
Arguments:
{String} MacroString * Строка проверяемая на макровставки.         RUS
Return:
{String} * Искомая строка с произведенной заменой найденый макровставок.       RUS
                    CONTENT ↑
Explication: Этот Метод  предназначен для обработки строкового значения указанного в качестве
какого либо пути. Формирование абсолютных путей из относительных.    
USM.PathValidation(TargetPath) {Method}
Arguments:
{String} TargetPath * Абсолютный или относительный путь к целевому файлу или каталогу. RUS
Внимание!!! Относительные пути вверх по лесу (древу каталогов), данная функция RUS
разбирает руководствуясь правлиом:  Каждая предстоящая точка кроме 
последней рассмативается как шаг вверх по лесу.   
Что отличается от схемы принятой в UNIX-подобных системах, где  
каждый последующий шаг верх по лесу преставляется конструкцией "/../"
Return:
{String} * Всегда абсолютный путь, полученный на основании полученного в качестве аргумента  RUS
относительного или абсолютного пути.            
                    CONTENT ↑
EXAMPLE
//  [ENG]                    
//  Above this code should be defined JScriptInclude Gear          
//  The "USM" module must be defined in config file of JScriptInclude Gear    
//  [RUS]                    
// Перед этим кодом должен быть обявлен механизм JScriptInclude Gear      
// модуль "USM"  должен быть определен в файле конфигурации JScriptInclude Gear    
                       
var somepath = "....\\Target.file"              
WScript.Echo(USM.PathValidation(somepath));            
//Для вызывающего скрипта который расположеного в C:\1\2\3\4\        
//вернет C:\1\Target.file                
                       
var somepath = "sometext_YYYY_MM_DD"              
WScript.Echo(USM.MacrosBase.MacrosParser(somepath));          
                       
//Вызванный 24.07.2012                  
//вернет "sometext_2012_07_24"
